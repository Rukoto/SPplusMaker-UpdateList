「SP+メーカー」用 "Word/Excel/PowerPoint ファイル形式用
    Microsoft Office 互換機能パック" 追加専用 アップデートリスト
_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/

「概要」
 　"Word/Excel/PowerPoint ファイル形式用 Microsoft Office 互換機能パック"
 を追加し、サービスパックとセキュリティ更新プログラムも一括導入する事のみを
 実現する為に作成したアップデートリストです。
 2017/07/05時点で最新の状態であること目標に作られています。
 時点の設定はHotfixのデジタル署名に基づいて定めています。
 （SP+メーカーのバージョンアップの際等に、例外的に時点は変わります。）

「対応OS」
 ・Windows 2000 Service Pack 4
 ・Windows XP Service Pack 1 以降
 ・Windows Vista
 ・Windows 7
 ・Windows 8
 ・Windows 10
 ・Windows Server 2003
 ・Windows Server 2008
 ・Windows Server 2008 r2
 ・Windows Server 2012
 ・Windows Server 2016

 　上記の32ビット版に対応しております。 64ビット版では、
 　WOW 64 サブシステム上での動作において対応しています。
 　動作保証外のOSでは互換性による不具合に注意してください。

「動作環境」
 　下記ソフトウェアが、最低でも1つインストールされている必要があります。
 ・Microsoft Word/Excel/PowerPoint 2000 Service Pack 3
 ・Microsoft Word/Excel/PowerPoint 2002 Service Pack 3
 ・Microsoft Word/Excel/PowerPoint 2003 Service Pack 1 以降
 ・Microsoft Word/Excel/PowerPoint Viewer 2003

「使用方法」
 　SP+メーカーの"Office Viewer"に対しての設定で、アップデートリストに
 本リストを選択することで使用することができます。
